Hoe kom ik aan een toegangstoken?

  1. Ga in Shopify naar Settings → Apps and sales channels → Develop apps.
  2. Maak een custom app aan (bijv. "Follow O-I").
  3. Geef onder Admin API scopes minimaal read_products rechten.
  4. Installeer de app en kopieer het Admin API access token (begint met shpat_).
  5. Plak het token hierboven, sla op, test de verbinding en klik op synchroniseren.

Het token wordt veilig opgeslagen en nooit op het scherm getoond. Synchroniseren werkt het bestaande aanbod bij en voegt nieuwe producten toe.
